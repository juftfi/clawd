# Integrations Use Cases Template

## Purpose
Show practical applications of an integration.

## Structure
1. **Hook** (1 sentence): Transformation this enables
2. **Use case 1** (2-3 sentences): Specific scenario
3. **Use case 2** (2-3 sentences): Different scenario
4. **Use case 3** (2-3 sentences): Power user scenario
5. **Start now** (1 sentence): CTA with setup link
