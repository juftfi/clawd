# Skills Spotlight Template

## Purpose
Highlight a single skill, its capabilities, and real-world applications.

## Structure
1. **Hook** (1 sentence): Problem this skill solves
2. **What it does** (2-3 sentences): Core functionality
3. **Example commands** (3-4 bullets): Natural language examples
4. **Use case** (1 paragraph): Real scenario where this shines
5. **Getting started** (1 sentence): Installation hint

## SEO Focus
- Primary: "[Skill Name] clawd skill"
- Secondary: "clawd [category] automation"

## Tone
Practical, benefit-focused, conversational

## Example Hook
"Tired of manually tracking every expense? The Expense Tracker Pro skill turns natural language into organized financial data."
