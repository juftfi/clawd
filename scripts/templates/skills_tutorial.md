# Skills Tutorial Template

## Purpose
Step-by-step guide to using a skill effectively.

## Structure
1. **Intro** (1 sentence): What you'll learn
2. **Prerequisites** (bullets): What you need
3. **Step 1-4** (numbered): Clear instructions with examples
4. **Pro tips** (2-3 bullets): Power user tricks
5. **Next steps** (1 sentence): Related skills or features

## SEO Focus
- Primary: "how to use [skill name]"
- Secondary: "clawd [action] tutorial"

## Tone
Instructional, friendly, encouraging

## Example Intro
"In this guide, you'll learn how to set up and master the Habit Tracker skill to build lasting routines."
