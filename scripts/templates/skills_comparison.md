# Skills Comparison Template

## Purpose
Compare similar skills or skill vs manual approach.

## Structure
1. **The choice** (1 sentence): What you're comparing
2. **Option A overview** (2-3 sentences): First approach
3. **Option B overview** (2-3 sentences): Second approach
4. **Comparison table** (3-5 rows): Key differences
5. **Verdict** (1-2 sentences): When to use each

## SEO Focus
- Primary: "[skill A] vs [skill B]"
- Secondary: "best clawd skill for [use case]"

## Tone
Objective, helpful, decisive

## Example Intro
"Both Portfolio Watcher and Expense Tracker Pro handle money, but they solve very different problems."
